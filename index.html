ROS in OSX with Docker - Installation

2016-10-02

Installing ROS in OSX from scratch is a pain.

The purpose of this setup is to create a boilerplate for the entire robotic development stack. Firstly we pull the official image from the registry, the size of this image is about 825 MB, so it would be better to make a light-weight one later on.

docker pull ros:indigo

or directly running

docker run --name ros-indigo -it ros:indigo

now you have a ROS environment, it works like a charm.
